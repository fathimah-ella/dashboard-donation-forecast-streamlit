This repository contains a web-based interactive dashboard for analyzing and forecasting donation transactions using Streamlit and Prophet. Built as part of a real-world internship project at Lazismu Banyumas, the dashboard processes donation data collected from the SobatBerbagi.com platform using both QRIS and manual transfer methods.It provides key insights into donor behavior, transaction trends, campaign performance, and donation method preferences. The dashboard also includes a time series forecasting model using Prophet to predict donation trends for the next 30 days, supporting data-driven fundraising strategies.
